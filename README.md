# 🐿️ 투자초보다람쥐 (Investment Novice Squirrel)
> **"내 돈 잃지 않고 배우는 안전한 가상화폐 투자 연습장"**
> 
> 빗썸(Bithumb) API 실시간 시세를 연동한 Java 기반 모의투자 플랫폼

## 1. 📖 프로젝트 개요
**투자초보다람쥐**는 투자를 시작하고 싶지만 높은 변동성과 원금 손실의 두려움으로 망설이는 초보자들을 위해 기획되었습니다. 
사용자는 가입 즉시 지급되는 가상 투자지원금(100만 원)과 매일 금융 퀴즈를 통해 얻은 보상금을 활용하여, **실제 시장 가격에 맞춰 위험 부담 없이 코인 매매를 연습**할 수 있습니다.

### 🛠️ 개발 환경 및 기술 스택
| 분류 | 기술 스택 |
| :-- | :-- |
| **Language** | Java 17 (Open JDK) |
| **Database** | Oracle 21c XE |
| **Connectivity** | JDBC (Java Database Connectivity) |
| **API** | Bithumb Public API (실시간 시세 조회) |
| **IDE** | IntelliJ IDEA |

## 2. 💡 핵심 기능
* **실시간 시세 연동**: 빗썸 API를 통해 비트코인(BTC), 이더리움(ETH), 도지코인(DOGE), 솔라나(SOL) 코인들의 실시간 가격을 반영합니다.
* **안전한 트랜잭션 처리**: 매수/매도 시 `투자금 갱신` - `포트폴리오 갱신` - `거래 기록 저장`의 3단계 과정을 하나의 트랜잭션으로 묶어 데이터 무결성을 보장했습니다.
* **오늘의 퀴즈 & 보상**: 금융 지식 퀴즈(일 1회)를 통해 학습을 유도하고, 정답 시 투자금 10만원을 추가로 지급합니다.
* **예외 처리 및 유효성 검사**: 로그인, 회원가입 시 잘못된 입력에 대해 프로그램을 종료하지 않고 재입력을 유도하는 무한 루프 로직을 구현했습니다.
